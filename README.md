# Admin Dashboard

A React + Vite based Admin Dashboard project.

## Features

- User Authentication
- View Users
- Add User
- Edit User
- Delete User
- Protected Routes

## Tech Stack

- React
- Vite
- React Router
- Axios
- JSON Server

## Installation

```bash
git clone https://github.com/AshokBhargav123/admin_dashboard.git
cd admin_dashboard
npm install
npm run dev
